# 图片消息MessageImage 

基础用法

```html
    <cg-message-image src="https://fastly.jsdelivr.net/npm/@vant/assets/cat.jpeg"></cg-message-image>
```

### Props

| 参数     | 说明                 | 类型    | 可选值                    | 默认值  |
| src | 图片地址 | string | ------------------------- | ------- |
| alt | 图片说明 | string | ------------------------- | ------- |
| width | 图片宽度 | number｜string | ------------------------- | ------- |
| height | 图片高度 | number｜string | ------------------------- | ------- |
| fit | object-fit | string | 'contain' 'cover' 'fill' 'none' 'scale-down' | ------- |
