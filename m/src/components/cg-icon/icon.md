## icon 图标 name="文件夹名字_图标名字"

基础用法

```javascript
 <cg-icon :name="icon_setvip"></cg-icon>
```

```javascript 可自定义宽高
<cg-icon :name="icon_setvip" :width="100"></cg-icon>

```

### Props

| 参数    | 说明                          | 类型   | 可选值              | 默认值 |
| ------- | ----------------------------- | ------ | ------------------- | ------ |
| name   | 图标名称 文件夹名字\_图标名字 | string | [list](./list.json) |        |
| width  | 图标宽度                      | string |                     | 40     |
| height | 图标高度                      | string |                     | 40     |
