# Toast

基础用法

```javascript
Toast('Đã gửi')
```

### Props

| 参数     | 说明                                          | 类型   | 可选值  | 默认值  |
| -------- | --------------------------------------------- | ------ | ------- | ------- |
| id       | 标识符，相同时共用一个实例，默认为多个实例    | string | -       | -       |
| type     | 展示类型                                      | string | success | success |
| duration | 展示时长（毫秒）值为 0 时，toast 不会自动消失 | number | -       | 2000    |

type 可扩展

### 返回值

| 参数       | 说明       | 类型       |
| ---------- | ---------- | ---------- |
| removeNode | 关闭 Toast | () => void |
